# getStockPrice

个人用的股票涨跌幅提醒小服务：接入老虎证券行情，通过飞书机器人推送通知，并可在飞书里管理监控列表。

## 简介

服务常驻后台运行，订阅你关心的股票实时报价。当价格波动达到你设定的幅度时，飞书会收到提醒；添加、删除股票和调整阈值，也可以在飞书里完成。

适合少量重仓股盯盘，不打算做成完整的交易系统。

## 技术栈

Python · 老虎证券 Open API · 飞书开放平台

## 快速开始

```bash
pip install -r requirements.txt
cp .env.example .env
# 填入老虎与飞书凭证后启动
python -m app.main
```

环境变量示例见 [.env.example](.env.example)。
